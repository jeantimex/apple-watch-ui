About this project

This is my second AngularJS project, the first one is 华容道(Hua Rong Dao). When Apple released the Apple Watch, I was quite interested in its UI design, so unique and elegant, I guess it must be written in Objective-C or Swfit plus Core Animation, as a JavaScript fanboi, my first thought is to build an Apple Watch UI with HTML5 and JavaScript.

Frameworks

Good frameworks can help scaffold and build HTML5 project easily, I choose Yeoman because it integrates tools like Grunt, Bower etc and it also works smoothly with AngularJS.

From my point of view, AngularJS is perfect for building small projects, it's easy to setup, easy to learn and it has a lot of great features like directive, service and factory.

I really enjoyed using Yeoman to scaffold AngularJS project, it makes my life so much easier. Also my favourite IDE is WebStorm, it integrates Grunt so that I can manage all the tasks(test, dist, serve etc) very easily.

Complete source

The complete source of this project can be found on Github, to build the project locally, clone the source and run:

$ npm install
$ bower install
$ grunt serve
